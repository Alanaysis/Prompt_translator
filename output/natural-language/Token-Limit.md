# Token Limit

**别名**: token 限制, 上下文限制

## 定义

**中文**: 模型在单次请求中能处理的最大 token 总数（输入 + 输出）

**English**: The maximum total number of tokens (input + output) a model can handle in a single request

> 💡 **Agent 提示**: Token Limit = max input + output tokens. 'token 限制'. Input tokens + max output tokens ≤ model limit. Plan accordingly.

## 示例

- "超过 token limit 了"
  → 输入和期望输出的总 token 数超出了模型限制
