# Latency (AI)

**别名**: 推理延迟, 首 token 延迟, TTFT, Time to First Token

## 定义

**中文**: 从发送请求到收到模型第一个响应 token 之间的时间延迟

**English**: The time delay between sending a request to an AI model and receiving the first token of the response

> 💡 **Agent 提示**: AI Latency = time to first token (TTFT). '推理延迟'. Different from throughput. Streaming reduces perceived latency.

## 示例

- "首 token 延迟太高了"
  → 模型开始响应的时间太长，用户体验差

## 关联术语

- throughput
- streaming
- token
