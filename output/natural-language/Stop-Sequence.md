# Stop Sequence

**别名**: 停止序列, 终止符, Stop Token

## 定义

**中文**: 告诉模型停止生成更多 token 的特定字符串

**English**: A specific string that tells the model to stop generating further tokens

> 💡 **Agent 提示**: Stop Sequence = string that halts generation. '停止序列'. Useful for structured output: stop at '```' after code block.

## 示例

- "加个 stop sequence"
  → 设置一个终止字符串，让模型在输出到特定标记时停止
