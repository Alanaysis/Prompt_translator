# Tool Use

**别名**: 工具调用, Function Calling, 工具使用

## 定义

**中文**: LLM 调用外部工具（API、计算器、数据库）以增强自身能力的特性

**English**: The ability of an LLM to invoke external tools (APIs, calculators, databases) to augment its capabilities

> 💡 **Agent 提示**: Tool Use = LLM calling external tools. '工具调用'. Extends capabilities beyond text. Examples: search, code execution, API calls.

## 示例

- "让模型调用搜索工具"
  → 给 LLM 提供搜索 API，让它能获取实时信息
