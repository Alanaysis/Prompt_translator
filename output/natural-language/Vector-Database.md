# Vector Database

**别名**: 向量数据库, 向量检索

## 定义

**中文**: 针对存储和搜索高维向量（嵌入）优化的数据库

**English**: A database optimized for storing and searching high-dimensional vectors (embeddings)

> 💡 **Agent 提示**: Vector DB = store and search embeddings. '向量数据库'. Tools: Pinecone, Milvus, Weaviate, ChromaDB. Key for RAG.

## 示例

- "用向量数据库做语义搜索"
  → 将文本向量化后存入专用数据库，支持基于语义相似度的搜索
