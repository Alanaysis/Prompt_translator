# System Prompt

**别名**: 系统提示词, 系统指令, System Message

## 定义

**中文**: 在对话开始前给 LLM 的特殊指令，定义其行为、角色和约束

**English**: A special instruction given to an LLM before the conversation starts, defining its behavior, persona, and constraints

> 💡 **Agent 提示**: System prompt = instruction before conversation. '系统提示词'. Sets persona, rules, constraints. Hidden from end users.

## 示例

- "在 system prompt 里加上约束"
  → 在系统提示词中定义 AI 的行为规则和限制
