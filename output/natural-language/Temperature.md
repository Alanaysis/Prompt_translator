# Temperature

**别名**: 温度, 随机性

## 定义

**中文**: 控制 LLM 输出随机性的参数。越低越确定，越高越有创造性

**English**: A parameter controlling the randomness of LLM output. Lower = more deterministic, Higher = more creative

> 💡 **Agent 提示**: Temperature = randomness control. '温度'. 0 = deterministic, 1 = creative, >1 = very random. Code: use low temp.

## 示例

- "temperature 调低一点"
  → 减少随机性，让输出更确定和一致
