# Max Tokens

**别名**: 最大 token 数, 最大长度, 输出长度限制

## 定义

**中文**: 模型在响应中生成的最大 token 数量

**English**: The maximum number of tokens the model will generate in its response

> 💡 **Agent 提示**: Max Tokens = output length limit. '最大 token 数'. Separate from context window. Set based on expected output length.

## 示例

- "max tokens 设大一点"
  → 增加输出长度限制，让模型生成更长的回答
