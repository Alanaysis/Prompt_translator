# Persona

**别名**: 用户画像, 用户角色, User Persona

## 定义

**中文**: 基于研究数据创建的、代表关键用户群体的虚构角色

**English**: A fictional character representing a key user segment, based on research data

> 💡 **Agent 提示**: Persona = fictional user representative. '用户画像'. Based on real research. Includes goals, frustrations, behaviors.

## 示例

- "我们的核心 persona 是什么？"
  → 我们的主要用户画像是什么样的人？

## 关联术语

- user-research
- segmentation
- UX
