# Activation

**别名**: 激活, 用户激活

## 定义

**中文**: 新用户首次体验到产品核心价值的时刻

**English**: The point where a new user first experiences the core value of a product

> 💡 **Agent 提示**: Activation = user first experiences core value. '激活'. Define your activation moment. Example: 'sent first message' for chat app.

## 示例

- "激活率太低了"
  → 注册的用户中真正使用了核心功能的比例太低
