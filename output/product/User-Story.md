# User Story

**别名**: 用户故事, User Story

## 定义

**中文**: 从最终用户角度描述功能的简短叙述（作为[用户]，我想要[目标]，以便[获得价值]）

**English**: A short, simple description of a feature told from the perspective of the end user (As a [user], I want [goal] so that [benefit])

> 💡 **Agent 提示**: User story = feature described from user perspective. Format: 'As a [role], I want [feature], so that [benefit]'.

## 示例

- "写个用户故事"
  → 用「作为X，我想要Y，以便Z」的格式描述需求
