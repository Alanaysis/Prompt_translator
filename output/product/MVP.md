# MVP

**别名**: 最小可行产品, 最小可用产品, Minimum Viable Product

## 定义

**中文**: 产品最简单的版本，可以发布以测试核心假设并收集用户反馈

**English**: The simplest version of a product that can be released to test core hypotheses and gather user feedback

> 💡 **Agent 提示**: MVP = minimum version to test if users want it. '最小可行产品'. NOT prototype (原型) - MVP is shipped to real users.

## 示例

- "先做 MVP 验证一下"
  → 先做最简单的版本发布出去，看看用户是否真的需要这个功能
