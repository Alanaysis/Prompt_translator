# Wireframe

**别名**: 线框图, 原型图, 低保真原型

## 定义

**中文**: UI 布局的低保真可视化表示，展示结构而非详细设计

**English**: A low-fidelity visual representation of a UI layout, showing structure without detailed design

> 💡 **Agent 提示**: Wireframe = low-fi UI sketch. '线框图'. Structure only, no colors/details. Tools: Figma, Balsamiq, paper sketch.

## 示例

- "先画个线框图"
  → 在开始设计前先用线框图确定页面布局和元素位置
