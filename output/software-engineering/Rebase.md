# Rebase

**别名**: 变基, Git Rebase

## 定义

**中文**: 将一个分支的提交重放到另一个分支的基础上，创建线性历史

**English**: Replaying commits from one branch onto the base of another, creating a linear history

> 💡 **Agent 提示**: Rebase = replay commits on new base. '变基'. Creates linear history. Don't rebase shared branches.

## 示例

- "rebase 到 main 上"
  → 把当前分支的提交重放到 main 分支的最新提交之后
