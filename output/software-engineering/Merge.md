# Merge

**别名**: 合并, Git Merge, 代码合并

## 定义

**中文**: 将一个分支的变更合并到另一个分支

**English**: Combining changes from one branch into another

> 💡 **Agent 提示**: Merge = combine branches. '合并'. Creates merge commit. vs Rebase = replay commits linearly.

## 示例

- "把 feature 分支合并到 main"
  → 将功能分支的代码变更合并回主分支
