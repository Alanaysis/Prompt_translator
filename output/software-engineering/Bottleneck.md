# Bottleneck

**别名**: 瓶颈, 性能瓶颈

## 定义

**中文**: 系统中限制整体性能或吞吐量的环节

**English**: The point in a system that limits overall performance or throughput

> 💡 **Agent 提示**: Bottleneck = weakest link limiting performance. '瓶颈'. Find with profiling, fix the slowest part first.

## 示例

- "数据库是瓶颈"
  → 系统的性能瓶颈在数据库层面
