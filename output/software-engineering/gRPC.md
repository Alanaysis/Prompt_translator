# gRPC

**别名**: gRPC 框架

## 定义

**中文**: 使用 Protocol Buffers 序列化和 HTTP/2 传输的高性能 RPC 框架

**English**: A high-performance RPC framework using Protocol Buffers for serialization and HTTP/2 for transport

> 💡 **Agent 提示**: gRPC = high-performance RPC with protobuf. Google's framework. Fast, typed, bidirectional streaming.

## 示例

- "内部服务间用 gRPC 通信"
  → 微服务之间用 gRPC 进行高性能的远程过程调用
