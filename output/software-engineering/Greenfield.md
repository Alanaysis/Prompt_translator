# Greenfield

**别名**: 绿地项目, 全新项目, 从零开始

## 定义

**中文**: 从零开始的项目，没有现有代码库或之前工作的约束

**English**: A project started from scratch with no existing codebase or constraints from prior work

> 💡 **Agent 提示**: Greenfield = start from nothing. Brownfield = work within existing system. '绿地'/'从零开始' = greenfield.

## 示例

- "这是个绿地项目"
  → 全新项目，可以从头选择技术栈和架构

## 关联术语

- legacy-code
