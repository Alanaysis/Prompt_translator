# Boilerplate

**别名**: 模板代码, 样板代码, 重复代码

## 定义

**中文**: 在项目间或项目内几乎不加修改就重复使用的代码

**English**: Code that is repeated with little or no alteration across projects or within a project

> 💡 **Agent 提示**: Boilerplate = repetitive code you always write. '模板代码'/'样板代码'. Reduce with frameworks, code generation.

## 示例

- "太多 boilerplate 了"
  → 有大量重复的模板代码，应该抽象或用工具生成
