# SRE

**别名**: 站点可靠性工程, Site Reliability Engineering

## 定义

**中文**: 将软件工程实践应用于基础设施和运维问题的学科

**English**: A discipline that applies software engineering practices to infrastructure and operations problems

> 💡 **Agent 提示**: SRE = software engineering approach to operations. '站点可靠性工程'. Google's approach. Error budgets, toil reduction.

## 示例

- "用 SRE 的方式管理服务"
  → 用软件工程的方法论来解决运维问题，关注可靠性指标
