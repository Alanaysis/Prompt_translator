# Feature Flag

**别名**: 功能开关, 特性开关, Feature Toggle, 功能旗标

## 定义

**中文**: 在运行时启用或禁用功能的机制，无需部署新代码

**English**: A mechanism to enable or disable features at runtime without deploying new code

> 💡 **Agent 提示**: Feature flag = runtime on/off switch for features. '功能开关'. Enables gradual rollout, A/B testing.

## 示例

- "用 feature flag 控制新功能的灰度"
  → 通过功能开关逐步向用户开放新功能

## 关联术语

- A/B testing
- canary-release
