# WebSocket

**别名**: WebSocket 连接, 长连接, WS

## 定义

**中文**: 在单个 TCP 连接上提供全双工通信通道的协议

**English**: A protocol providing full-duplex communication channels over a single TCP connection

> 💡 **Agent 提示**: WebSocket = persistent bidirectional connection. '长连接'. For real-time: chat, live data, gaming. vs HTTP polling.

## 示例

- "用 WebSocket 做实时推送"
  → 建立长连接，服务端可以主动向客户端推送消息
