# Singleton

**别名**: 单例模式, 单例

## 定义

**中文**: 确保一个类只有一个实例，并提供全局访问点的设计模式

**English**: A design pattern that ensures a class has only one instance and provides a global point of access to it

> 💡 **Agent 提示**: Singleton = one instance only. '单例'. Use sparingly, often considered an anti-pattern for testability.

## 示例

- "数据库连接用单例模式"
  → 整个应用只维护一个数据库连接实例
