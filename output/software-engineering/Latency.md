# Latency

**别名**: 延迟, 响应时间, 响应延迟, Response Time

## 定义

**中文**: 从发起请求到收到响应之间的时间延迟，通常以毫秒为单位

**English**: The time delay between a request and its response, typically measured in milliseconds

> 💡 **Agent 提示**: Latency = delay/响应时间. '延迟高' = high latency = slow response. Not to be confused with throughput (吞吐量).

## 示例

- "接口延迟太高了"
  → API 响应时间太长，用户需要等很久

- "p99 延迟 200ms"
  → 99% 的请求在 200ms 内返回
