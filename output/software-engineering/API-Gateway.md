# API Gateway

**别名**: API 网关, 网关

## 定义

**中文**: 所有 API 请求的统一入口，处理路由、认证、限流等

**English**: A single entry point for all API requests, handling routing, authentication, rate limiting, etc.

> 💡 **Agent 提示**: API Gateway = single entry point for all APIs. 'API 网关'. Handles auth, rate limiting, routing. Key in microservices.

## 示例

- "在前面加个 API 网关"
  → 统一入口处理认证、限流、路由等横切关注点

## 关联术语

- microservices
- middleware
- load-balancer
