# JWT

**别名**: JSON Web Token, 令牌, Token

## 定义

**中文**: 一种紧凑、URL 安全的令牌格式，用于在各方之间安全传输 JSON 对象信息

**English**: A compact, URL-safe token format for securely transmitting information between parties as a JSON object

> 💡 **Agent 提示**: JWT = JSON Web Token for auth. '令牌'. Stateless, contains claims, signed not encrypted.

## 示例

- "用 JWT 做身份验证"
  → 用户登录后发放 JWT，后续请求携带此令牌验证身份
