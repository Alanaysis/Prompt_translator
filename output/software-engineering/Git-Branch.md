# Git Branch

**别名**: 分支, Git 分支, Branch

## 定义

**中文**: 版本控制中独立的开发线路

**English**: An independent line of development in version control

> 💡 **Agent 提示**: Branch = independent development line. '分支'. Strategies: Git Flow, GitHub Flow, Trunk-based.

## 示例

- "拉个新分支开发"
  → 从主分支创建一个新分支，在上面开发新功能

## 关联术语

- merge
- rebase
- git-flow
