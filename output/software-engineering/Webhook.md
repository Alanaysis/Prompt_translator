# Webhook

**别名**: 回调, 钩子, 回调通知

## 定义

**中文**: 由一个系统的事件触发的 HTTP 回调，向另一个系统的 URL 发送数据

**English**: An HTTP callback triggered by an event in one system, sending data to another system's URL

> 💡 **Agent 提示**: Webhook = HTTP callback on event. '回调'/'钩子'. Push-based, not pull. Like a reverse API call.

## 示例

- "配个 webhook 通知我们"
  → 当事件发生时，对方系统会向我们的 URL 发送 HTTP 请求通知
