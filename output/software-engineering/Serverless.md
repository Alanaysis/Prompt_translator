# Serverless

**别名**: 无服务器, Serverless 架构, FaaS, 函数计算

## 定义

**中文**: 云服务商管理基础设施、你只需按实际执行时间付费的云计算模型

**English**: A cloud model where the provider manages infrastructure and you only pay for actual execution time

> 💡 **Agent 提示**: Serverless = no server management, pay per execution. '无服务器'. Examples: AWS Lambda, Cloudflare Workers.

## 示例

- "用 serverless 部署"
  → 不需要管理服务器，按调用次数付费

## 关联术语

- cloud-native
- auto-scaling
- Lambda
