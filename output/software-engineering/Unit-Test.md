# Unit Test

**别名**: 单元测试, 单测

## 定义

**中文**: 单独测试各个函数或方法以验证其正确性

**English**: Testing individual functions or methods in isolation to verify they work correctly

> 💡 **Agent 提示**: Unit test = test individual functions in isolation. '单元测试'. Fast, cheap, many. Testing pyramid base.

## 示例

- "写个单测覆盖这个函数"
  → 为这个函数编写单元测试，验证输入输出
