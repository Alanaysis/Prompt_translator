# Infrastructure as Code

**别名**: 基础设施即代码, IaC

## 定义

**中文**: 通过机器可读的配置文件（而非手动操作）来管理基础设施

**English**: Managing infrastructure through machine-readable configuration files rather than manual processes

> 💡 **Agent 提示**: IaC = infrastructure as code, not manual setup. '基础设施即代码'. Tools: Terraform, Pulumi, CloudFormation.

## 示例

- "用 Terraform 做 IaC"
  → 用代码定义和管理云基础设施，可版本化、可复现

## 关联术语

- DevOps
- containerization
- cloud-native
