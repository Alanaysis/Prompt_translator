# Integration Test

**别名**: 集成测试, 联调测试

## 定义

**中文**: 测试多个组件协同工作的正确性

**English**: Testing that multiple components work together correctly as a group

> 💡 **Agent 提示**: Integration test = test components working together. '集成测试'. Slower than unit tests, catches interface issues.

## 示例

- "跑一下集成测试"
  → 测试各模块之间的交互是否正常

- E2E-testing
