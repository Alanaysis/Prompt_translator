# Database Migration

**别名**: 数据库迁移, Migration, DB Migration, 数据迁移

## 定义

**中文**: 对数据库结构的版本化变更（添加表、列、修改结构）

**English**: Version-controlled changes to database schema (adding tables, columns, modifying structure)

> 💡 **Agent 提示**: Migration = version-controlled DB schema change. '数据库迁移'. Tools: Alembic, Flyway, Rails migrations.

## 示例

- "跑一下 migration"
  → 执行数据库结构变更脚本
