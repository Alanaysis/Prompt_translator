# Conflict

**别名**: 冲突, 合并冲突, Merge Conflict

## 定义

**中文**: 两个分支修改了同一行代码，Git 无法自动合并

**English**: When two branches modify the same lines of code and Git cannot automatically merge them

> 💡 **Agent 提示**: Conflict = can't auto-merge, needs manual resolution. '冲突'. Common in team development.

## 示例

- "有冲突，需要手动解决"
  → 合并时同一文件被不同分支修改，需要人工决定保留哪个版本

## 关联术语
