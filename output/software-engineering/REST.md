# REST

**别名**: RESTful, REST API, RESTful API

## 定义

**中文**: 使用标准 HTTP 方法（GET、POST、PUT、DELETE）和基于资源的 URL 的 API 架构风格

**English**: An architectural style for APIs using standard HTTP methods (GET, POST, PUT, DELETE) and resource-based URLs

> 💡 **Agent 提示**: REST = HTTP-based API style. Resources + HTTP methods. GET=read, POST=create, PUT=update, DELETE=delete.

## 示例

- "用 REST 风格设计 API"
  → 用 HTTP 方法映射 CRUD 操作，URL 表示资源
