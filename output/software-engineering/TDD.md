# TDD

**别名**: 测试驱动开发, Test-Driven Development

## 定义

**中文**: 先写测试再写实现代码的开发方法（红 → 绿 → 重构）

**English**: A development approach where you write tests before writing the implementation code (Red → Green → Refactor)

> 💡 **Agent 提示**: TDD = write test first, then code. '测试驱动开发'. Cycle: Red (fail) → Green (pass) → Refactor.

## 示例

- "用 TDD 的方式开发"
  → 先写失败的测试，再写代码让测试通过，最后重构

## 关联术语
