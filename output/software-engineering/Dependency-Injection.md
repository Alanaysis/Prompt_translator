# Dependency Injection

**别名**: 依赖注入, DI, IoC

## 定义

**中文**: 对象从外部接收其依赖项，而不是在内部自行创建的技术

**English**: A technique where an object receives its dependencies from external sources rather than creating them internally

> 💡 **Agent 提示**: DI = don't create dependencies yourself, receive them from outside. Promotes testability and loose coupling.

## 示例

- "用依赖注入来解耦"
  → 通过外部注入依赖而不是内部创建，来降低组件间的耦合度
