# GitOps

**别名**: Git 运维

## 定义

**中文**: 以 Git 作为基础设施和应用交付唯一事实来源的部署范式

**English**: A deployment paradigm where Git is the single source of truth for infrastructure and application delivery

> 💡 **Agent 提示**: GitOps = Git as single source of truth for infra. 'Git 运维'. Declarative, version-controlled, automated.

## 示例

- "用 GitOps 管理部署"
  → 所有基础设施和应用配置都在 Git 中，变更通过 PR 触发自动部署
