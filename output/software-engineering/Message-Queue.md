# Message Queue

**别名**: 消息队列, MQ, 消息中间件

## 定义

**中文**: 通过排队消息实现服务间异步通信的系统

**English**: A system for asynchronous communication between services through queued messages

> 💡 **Agent 提示**: Message Queue = async communication between services. '消息队列'. Examples: RabbitMQ, Kafka, Redis Streams.

## 示例

- "用消息队列解耦"
  → 通过 MQ 异步处理，减少服务间的直接依赖

## 关联术语

- event-driven
- decoupling
- async
