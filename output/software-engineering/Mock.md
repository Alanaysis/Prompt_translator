# Mock

**别名**: 模拟对象, Mock 对象, Stub

## 定义

**中文**: 在测试中模拟真实依赖行为的假对象

**English**: A fake object that simulates the behavior of a real dependency in tests

> 💡 **Agent 提示**: Mock = fake object for testing. '模拟对象'. Replaces real dependencies. Mock vs Stub: mock verifies behavior, stub provides data.

## 示例

- "把数据库调用 mock 掉"
  → 用假对象替代真实数据库调用，让测试更快更可控
